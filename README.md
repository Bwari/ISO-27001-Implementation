# ISO-27001-Implementation

<h2>Status: In Progress</h2>
<b>Frameworks: </b>ISO/IEC 27001:2022, Kenya Data Protection Act 2019
<b>Tools & Methods: </b> Risk registers, Gap analysis, ISMS documentation, Control Mapping, POA&M

This project involves the end-to-end implementation of an ISO/IEC 27001 Information Security Management System (ISMS) for a mock SACCO based in Kenya. The goal is to simulate a real-world compliance engagement, addressing the unique information security challenges faced by financial cooperatives operating in the Kenyan regulatory environment — including alignment with the Data Protection Act 2019 and CBK/SASRA guidelines.

The project covers key ISMS phases including organisational context analysis, stakeholder identification, gap assessment against ISO 27001:2022 controls, risk assessment and treatment, development of security policies and procedures, and the design of a Statement of Applicability (SoA). The mock Sacco environment is used to demonstrate practical application of controls across areas such as access management, asset classification, incident response, and business continuity.
